Globall Cloud is a modern, secure, and scalable cloud platform built with React, TypeScript, and Node.js. Designed for performance, security, and PWA support.

### ✨ Live Demo & Release
Latest Stable Release: **Globall Cloud v1.0.0** - Check [Releases](../../releases)

### 🚀 Features
- **Admin Panel** - Complete management dashboard with RTL/LTR support
- **Backend API** - Secure RESTful API with Auth & Firestore Rules
- **Security First** - GDPR Ready, OWASP Top 10 Protected, PCI-DSS Compliant
- **PWA Ready** - Offline support, localStorage XSS protection, CORS secured API
- **Performance** - Code splitting, lazy loading, optimized build

### 🛠️ Tech Stack
- **Frontend:** React, TypeScript 85.1%, JavaScript 8.1%, CSS 5.6%, HTML 1.2%
- **Backend:** Node.js, Express
- **Security:** Firestore Rules, Auth Bypass Protection, UTF-8 Validation
- **DevOps:** GitHub Actions, Dependabot
